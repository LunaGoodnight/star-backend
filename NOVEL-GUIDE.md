# How to Continue Novels with Claude Code

## Quick Commands

### Continue a novel
Just say:
```
繼續寫萬界藥神
```
or
```
繼續寫星辰藥師
```
or
```
繼續寫灰鴉學院
```
or
```
繼續寫十君令
```
Claude will read the existing chapters from the API and continue the story.

### Generate more chapters at once (batch mode, no confirmation)
```
繼續寫萬界藥神 30篇，全部寫好一次發佈，不要問我確認。
繼續寫十君令 20篇，全部寫好一次發佈，不要問我確認。
```

### Generate other types of articles
```
幫我生十篇生活知識的文章
幫我寫五首詩
再來十個冷笑話
```

### Start a brand new novel
```
幫我開一個新的連載小說，題材是科幻的
```

### Check what articles exist
```
幫我列出所有文章
```

## Tips
- You can speak Chinese — Claude will understand and reply in English (with Chinese translations when needed)
- No need to give technical details — just say what you want in plain words
- Claude remembers the novel plots across sessions (saved in memory files)
- Each session has a daily token limit — ~20-30 articles per session is safe
