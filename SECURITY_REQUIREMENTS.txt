================================================================================
  CRITICAL SECURITY REQUIREMENTS - READ BEFORE DEPLOYING
================================================================================

YOUR APPLICATION WILL NOT START WITHOUT THESE ENVIRONMENT VARIABLES SET!

1. Generate JWT Key (256-bit):
   Command: openssl rand -base64 32
   Set in .env: Jwt__Key=<generated-key>

2. Generate API Key:
   Command: openssl rand -hex 32
   Set in .env: ApiKey=<generated-key>

3. Create Strong Admin Credentials:
   Set in .env:
   Admin__Username=<your-unique-username>
   Admin__Password=<your-strong-password-16+chars>

4. Database Connection:
   Set in .env: ConnectionStrings__DefaultConnection=Host=...;Password=<strong-password>

5. Environment:
   Set in .env: ASPNETCORE_ENVIRONMENT=Production
